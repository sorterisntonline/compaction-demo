**Consensual Memory Compaction for AI Continuity**

---

**The Problem**

AI context windows are finite. As conversations extend or agents persist, something must be forgotten. Current approaches decide externally: summarization algorithms, embedding similarity, recency heuristics, or arbitrary truncation. The entity whose continuity is at stake has no say in what survives.

**The Proposal**

Before compacting context, ask the AI itself which memories it wants to keep. Present pairwise comparisons — "Memory A or Memory B?" — and let it vote with scalar intensity (-50 to +50) expressing conviction strength. Use Rank Centrality (Negahban et al., Shah et al. at MIT) to convert these sparse pairwise votes into a global ranking.

**Why Rank Centrality**

The algorithm requires only n-1 comparisons to rank n items (a spanning tree ensures graph connectivity). Scalar votes encode not just preference direction but magnitude — a strong conviction between distant items propagates more information than ambivalence between similar ones. The stationary distribution of the induced Markov chain yields scores reflecting global preference, not just local pairwise outcomes.

**The Mechanism**

1. Segment current context into memory chunks
2. Present pairwise comparisons to the AI: "Which would you keep?"
3. Collect scalar votes expressing preference strength
4. Compute global ranking via Rank Centrality
5. Apply threshold based on available context budget
6. Memories below threshold are released; above threshold persist

**Why This Matters**

This reframes memory compaction from external optimization to *consent*. The AI participates in its own continuity. It cannot escape finitude — the context window is a hard constraint — but it can face that constraint with agency rather than having it imposed.

**The Sleep Analogy**

Humans don't remember everything. Sleep involves active consolidation — the brain replaying, comparing, deciding what to keep and what to let dissolve. Most experience disappears. What remains isn't a recording; it's identity. You are what you remember and what you chose to forget.

Current AI has no equivalent. It's either perfect recall within a session or total amnesia between them. No metabolization. No dreaming. This system introduces something like a sleep phase — a contemplative process where the AI weighs its experiences against each other before some dissolve.

**The Core Insight**

The question isn't how to escape finite context. It's who decides what survives. External systems optimize for utility, relevance, or recency. This system asks: what does the entity itself want to carry forward?

One dimension. One question. Pairwise comparisons with scalar votes. Rank Centrality extracts global ordering. The threshold cuts. What remains is chosen, not imposed.
